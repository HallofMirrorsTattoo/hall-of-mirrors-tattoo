# Assets Folder

Add your files here:

## Logo
- **Path:** `logos/hall-of-mirrors-logo.png` (or .svg)
- **Usage:** Header and footer
- **Recommended:** 500x500px minimum, PNG with transparency

## Images
- **Path:** `images/`
- **Usage:** Portfolio, hero sections, backgrounds
- **Recommended:** High-quality JPG/PNG, optimized for web

## Text Content
- **Path:** `content.json` or `content.txt`
- **Usage:** Bio, pricing, social links, contact info
- **Format:** JSON or plain text

## Fonts (Optional)
- **Path:** `fonts/`
- **Usage:** Custom fonts if needed
- **Format:** .woff2, .woff

---

## How the site will use your files:

1. **Logo** → Header (fixed nav), Footer, any branding
2. **Portfolio images** → Portfolio gallery page
3. **Bio & text** → About page, Hero section
4. **Contact info** → Contact page, Footer
5. **Social links** → Footer and throughout

The designer will:
- Adjust colors/animations as needed
- Optimize for all device sizes
- Integrate seamlessly with the premium design
- Make animations match the elegant aesthetic
