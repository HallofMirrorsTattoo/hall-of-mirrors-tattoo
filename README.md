# Hall of Mirrors Tattoo

Booking platform for Hall of Mirrors Tattoo Studio — Suite 3, 34 Castle Street, Liverpool L2 0NR.

Built with Next.js 14 + Express.js + Supabase PostgreSQL. Deployed on Vercel (frontend) + Railway (backend).
